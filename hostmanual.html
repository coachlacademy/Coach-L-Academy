6. Going Live (Host Workflow)

  1. Enter the Host Studio using one of the host passwords
  2. WebDJ loads automatically (AutoDJ pauses)
  3. Scroll down to the Live Video Chat window
  4. Click Join in Browser
  5. Enter your name (Host / DJ name)
  6. Allow microphone and camera access
  7. You are now live on air

8. Live Chat & Communication

  • Use the built-in chat inside the Live Video window
  • Chat allows:
    • Private coordination with guests
    • Reading listener comments (if enabled)

9. Ending a Show

  • End the Live Video Chat
  • WebDJ releases control back to AutoDJ
  • Return to standby automatically
  • Click Exit Studio if finished

10. Best Practices

  • Always test microphone before going live
  • Use headphones to avoid echo
  • Keep credentials private
  • Monitor schedule to avoid overlap
